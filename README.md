# design.metatooth.com

Dental CAD with Vue.js & Three.js

## Getting Started

### Prerequisites

#### Ubuntu 20.04

```
$ sudo apt-get update
$ sudo apt-get install nodejs npm
```

### Install

```
$ git clone https://github.com/metatooth/design.git
$ cd design
$ npm install
$ npm run serve
```
